    Cascadence (Ruby)

    Functional integration testing is suffering. It's hard to pass, hard to maintain, hard to run, and hard to sit through. Cascadence solve any of those problems, but it does make writing, running, and maintaining the tests easier by structuring tests to follow a merge / fork flow based perspective. This allows for easy multithreading and concurrency for running integration tests.

    $ gem install cascadence
OR
$ git clone git@github.com:foxnewsnetwork/cascadence.git
$ cd cascadence && rake install

    Openstory (RPGMaker IV & Ruby)

    Video games these days are all about flashy graphics or neat gameplay; gamers and game-makers alike nowadays gaffle and gasp over how innovative and cool some new game feature or how spiffy the character costume design looks. Graphics and toys are nice, but gone are the good old days when us story tellers should to weave color onto the blank canvas of the gamer's imagination with nothing more than words. Gone are the days when characters drew us in with nothing more than their thoughts, actions, and personalities.

    Gone, these things might be from the mainstream industry, but it's certainly not gone from this budding project. Note, not to be a pleb-tier hater, but you'll need a windows machine if you want to run this.

    git clone git@github.com:foxnewsnetwork/openstory.git

    Autofold (Ruby)

    rails scaffold is great, but sometimes, what if I want to scaffold more than just 1 resource at a time? What if I wanted to scaffold simply related resources like users that have many accounts and items which have many offers? Autofold interprets a simply expression and builds your web assets for you so you can spend your time being unemployed.

    $ gem install autofold
OR
$ git clone git@github.com:foxnewsnetwork/autofold.git
$ cd autofold && rake install

$ autofold generate 'users:(accounts+items:offers)'

    Tsundere (ruby)

    Yes, it's another permission gem for Ruby... worse yet, this one is influenced by category theory and haskell. Strictly speaking, tsundere is an implementation of haskell's Arrow typeclass and, when attached to an object, implements haskell's monad typeclass. It provides a clean and elegant but pointlessly complicated and mathy way of dealing with permission.

    In common terms, it exposes only portions of objects that certain users with correct permissions may access.

    $ gem install tsundere
